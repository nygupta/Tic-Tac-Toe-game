# TIC-TAC-TOE Game :--

This TicTacToe Game is build with react.js


## Local Development 

Clone the Repository 
```sh
$ git clone https://github.com/nygupta/TicTacToe-game
```
Install the dependencies
```sh
$ npm install
```
Run the server - this will spawn a development server with port `1234`.
```sh
$ npm run start
```
Build the server - this will output a production build in the `dist` directory.
```sh
$ npm run build
```

## Deployment 
The Game is deployed to [tictactoe-blacklion.surge.sh](https://tictactoe-blacklion.surge.sh/).
